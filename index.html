Machine Learning.

Create, train and deploy Machine Learning models.
Without expensive hardware.


Train and deploy Machine Learning models without the hassle.

Here's how it works:

1. Upload Data

Mlblocks allows you to create machine learning models with only 50 images per class. Collect your data and upload it in a click

2. Train Model

After you upload your data, we handle the rest! We'll train the model on our servers and get it back to you ASAP.

3. Deploy In A Click

Deploy and serve your models from the cloud without touching a single line of code!

4. Seamless integration

Easily integrate your trained models using a REST URL for instantaneous predictions!

With a simple HTTP request, have an entire Machine Learning suite of tools at the tips of your fingers!

import requests
from PIL import Image

img = Image.open('my_img.jpg')

url = 'mlblocks.com/XXXXXXXX/predict'
options = {
data: img,
api_key: ACCESS_TOKEN
}

prediction = requests.get(url, data= options)
